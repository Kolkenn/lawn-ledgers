# Install nvm
```curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash```
Check Github for latest script. https://github.com/nvm-sh/nvm

# Install LTS Version of Node.js and npm
```nvm install --lts --latest-npm```

# Install dependecies
```npm install```

# Setup Environment Variables.
1. Reference .env.example
2. Create a new .env file and gather the required information from the firebase console.

# Run the Web Server
```npm run dev```
